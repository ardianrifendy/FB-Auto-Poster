# FB-Auto-Poster 🚀

Alat otomatisasi untuk memposting konten ke Facebook (Marketplace/Groups) secara massal dengan fitur manajemen campaign yang cerdas.

## ✨ Fitur Utama
- **Web Dashboard**: Pantau dan kontrol proses posting melalui antarmuka web.
- **Spintax Support**: Variasi teks otomatis untuk menghindari deteksi spam.
- **Multi-Campaign**: Kelola banyak draft postingan sekaligus.
- **Smart Scraper**: Ekstraksi data dan interaksi otomatis dengan elemen Facebook.
- **Local Database**: Penyimpanan history postingan yang sukses maupun gagal.

## 🛠️ Teknologi
- **Node.js** & **Express** (Backend)
- **Puppeteer** (Automation Engine)
- **Vanilla JS & CSS** (Frontend Dashboard)

## 🚀 Cara Penggunaan
1. Clone repository ini.
2. Jalankan `npm install`.
3. Atur konfigurasi di `config.js`.
4. Jalankan bot dengan `node index.js` atau gunakan `MULAI_BOT.bat`.
